# AIエージェント設定代行

OpenClaw・Claude・Claude Code の導入支援サービスのランディングページ

## 技術スタック

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 でアクセス

## デプロイ

Vercel へのデプロイを推奨

```bash
npm run build
```
